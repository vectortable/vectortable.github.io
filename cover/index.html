A PDF scrolling, a screen recording, .  Player can only play against another player not against machine.  However, app will observe all standard chess pieces movement rules.  Those included basic, castling, pawn promotion, and en passant moves.

There are four game modes:
Game mode 1:  -  Play On A Single iDevice  -  Two players play using a single iDevice.
Game mode 2:  -  Play On Two iDevices Via Local Wifi  -  Two players play with individual iDevice via local WiFi.
Game mode 3:  -  Online Live Via Game Center  -  Two players play against each other live across Internet via Game Center.
Game mode 4:  -  Turn-based Via Game Center  -  Two players take turns  -  not live  -  across Internet via Game Center.
